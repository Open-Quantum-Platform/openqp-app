    Direct local run
    
            
    Set up OpenQP on this computer
    
          
    
          
        
    
        
    
          The local runner is a small program on this computer. Its pairing code lets this page send the prepared input to that runner without uploading the job.
        
    
        
    
          Start the local runner, then check it from this page.
        
    
        
      
          
    1. 
            Install OpenQP if the command is missing.
            
      Download the installer, then run it once from Terminal. It installs OpenQP from the official source repository into a local Python environment.
      
            
      
              Download OpenQP installer
              
            
      
            
      python3 ~/Downloads/install-openqp-local.py
      
          
      2. 
          
    2. 
            Start the local runner.
            
      Download the runner and launch it from Terminal. Keep that Terminal window open while jobs are running.
      
            
      
              Download local runner
              
            
      
            
      python3 ~/Downloads/openqp-local-runner.py
      
          
      3. 
          
    3. 
            Copy the pairing code.
            
      The runner prints a line like Pairing code: .... Paste that code into the Pairing code field on this page.
      
          
      4. 
          
    4. 
            Open local mode.
            
      Use local mode when the secure web page cannot reach your local runner directly.
      
            Open local mode
          
      5. 
          
    5. 
            Check, then run.
            
      Click Check local runner. If it finds OpenQP, click Run with local OpenQP. Results are written under your OpenQP Local Runs folder.
      
            
          
      6.
